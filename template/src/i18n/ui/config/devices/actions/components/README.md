# Example Device Action
This directory contains an example device action to show how you would build device actions associated with your service.  If you service has no devices, you don't need to do anything here.

## Files
* **device-action0.component.html** - The HTML template used by the example implementation.
* **device-action0.component.ts** - The Angular 9 Component that implements the device action.  It demonstrates what is provided as inputs.
